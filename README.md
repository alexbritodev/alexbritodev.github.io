# Alex Brito - Personal Website

🌐 A bilingual (EN/PT-BR) premium landing page focused on institutional presence, performance, and visual clarity.

## ✨ Overview

This project was created to represent **Alex Brito's personal brand** with an elegant, objective, and fully responsive experience across **desktop, tablet, and mobile**.

The website includes:

- English version: `index.html`
- Brazilian Portuguese version: `br/index.html`
- Visual identity with logo, favicon, and language flags
- Mobile flyout menu with close behavior via `X`, overlay click, menu item click, and `Esc`
- Semantic structure with SEO and Open Graph metadata

## 🧰 Stack

- `HTML5`
- `Tailwind CSS` (CDN)
- Custom `CSS` (premium UI layer)
- Vanilla `JavaScript`
- `Lucide Icons` (CDN)
